Необходимо иметь Node.js >= 18

Сборка верстки:
1) Перейти в каталог с версткой
2) "npm install" - для установки зависимостей
3) "npm run build" - для сборки (папка assets в корне проекта)
   "npm start" - для разработки (доступ в браузере)

После запуска "npm run build" сборка должна появиться в папке "assets" в корне проекта
После запуска "npm start" доступ в браузере: http://localhost:3000/pages.html - все страницы
