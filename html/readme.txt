Необходимо иметь Node.js >= 18

Сборка верстки:
1) Перейти в каталог с версткой "/html"
2) "npm install" - для установки зависимостей
3) "npm run build" - для сборки (папка "assets" в корне проекта)
   "npm start" - для разработки (доступ в браузере)

После запуска "npm run build" сборка должна появиться в папке "assets" в корне проекта.
После запуска "npm start" доступ в браузере: http://localhost:3000/pages.html - все страницы.
Если указать в "webpack.config.js" опцию "writeToDisk: true", то в режиме разработки фалйы будут записываться на диск в папку "assets", а не храниться в памяти.

Примечания:
1) В проект добавлен Stimulus (раздел "Тестирование").
   - Расположение: "js/stimulus"
   - Документация по Stimulus: https://stimulus.hotwired.dev/reference/controllers
   - Преимущества: инициализация js происходит сразу после добавления элемента DOM на страницу; упорядоченность.
